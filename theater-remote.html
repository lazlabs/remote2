Movie Room Remote · v2.2.0
Help & About
Two modes, zero confusion

This remote controls your entire theater room with two big buttons. Pick your mode and the room sets itself up.

📺 Casual TV

TV turns on, switches to HDMI 1 (Fire TV), and your Sonos soundbar handles audio automatically via eARC. The Yamaha stays off. Great for daily watching.

🎬 Movie Night

TV turns on, switches to HDMI 2, the Yamaha receiver powers up for 7.2 surround sound, Sonos is muted, and the room lights dim to 10%. Full cinema experience.

Volume

Both Sonos and Yamaha sliders are always visible so any family member can adjust either zone without switching modes.

Remote pad

Tap Remote in the bottom bar for the D-pad. Toggle between Fire TV and Gaming PC at the top — commands route to the right device automatically.

Nothing happens when I tap a mode button.
Check that your HA URL and token are configured in Settings. Make sure your phone/tablet is on the same network as Home Assistant. Use "Ping HA" in Manual Configuration to verify connectivity.
The OAuth Connect button redirects but comes back with an error.
The app must be hosted on HTTPS for OAuth to work. If you're opening the HTML file directly (file://) use a manual token instead. On GitHub Pages this works automatically.
Volume sliders move but nothing changes.
Confirm the entity IDs match exactly what HA shows. Go to HA → Settings → Devices & Services → your device, and copy the entity ID from there into Manual Configuration.
How do I add this to my home screen?
On iPhone/iPad: tap Share → Add to Home Screen. On Android: tap the browser menu → Add to Home Screen / Install App. It installs as a full-screen PWA with no browser chrome.
The D-pad doesn't control my Fire TV.
You need either the Fire TV integration or the Android TV / ADB integration in HA, with a remote. entity. The commands use remote/send_command. Check your entity ID prefix in HA's developer tools.

Setup Help
Finding your Home Assistant URL
1
Open Home Assistant on the same network
On any device connected to your home Wi-Fi, open a browser and try typing: homeassistant.local:8123
2
If that doesn't load, find the IP address
Go to your HA dashboard → Settings → System → Network. You'll see an IP address like 192.168.1.45. Your URL is then http://192.168.1.45:8123
3
Copy that URL into the field
Include http:// at the start and :8123 at the end. No trailing slash.
Using Nabu Casa? If you pay for Home Assistant Cloud, your URL looks like https://XXXXX.ui.nabu.casa — use that instead. It works over the internet too.
This app only works on your home Wi-Fi (or VPN) unless you're using Nabu Casa or a remote access setup. That's normal and keeps your home system secure.
Creating a Long-Lived Access Token
1
Open your HA profile
In Home Assistant, click your name or avatar in the very bottom-left of the sidebar. This opens your Profile page.
2
Scroll to the bottom
Scroll all the way down until you see a section called Long-Lived Access Tokens.
3
Create a token
Click Create Token. Type a name like Movie Room Remote and click OK.
4
Copy it immediately
HA shows the token once only. Copy the whole thing — it's a long string of letters and numbers. Paste it into the Token field in this app.
Important: If you close the HA dialog without copying, the token is gone and you'll need to create a new one. That's fine — just delete the old one and make another.
The token is like a password for this app. Keep this app's config backed up (Settings → Export Config) so you don't have to regenerate it.
Download ready-made automation YAML
The YAML file has all four automations pre-written. In HA go to Settings → Automations → ⋮ menu → Import from YAML and paste each block. You only need to update the entity IDs at the top of each block to match your devices.
What is a webhook?
A webhook is a web address that triggers an action when visited. When you tap Casual TV in this app, it sends a signal to that address. Home Assistant listens for that signal and runs whatever actions you set up — turning on the TV, switching inputs, adjusting audio.
Create your first automation — Casual TV
1
Go to Automations
In HA: Settings → Automations & Scenes → Create Automation → Create new automation
2
Add a Webhook trigger
Click Add Trigger → search for Webhook → select it. You'll see a field called Webhook ID. Type something simple with no spaces: casual_tv_on
3
Add your actions
Click Add Action for each thing you want to happen:
• Call service: media_player.turn_on → pick your TV
• Call service: media_player.select_source → TV → source: HDMI 1
• Call service: media_player.turn_off → pick your Yamaha
4
Save and copy the Webhook ID
Click Save in the top right. The Webhook ID you typed (casual_tv_on) is what you paste into this app. That's it.
Create Movie Night automation
Repeat the same steps with Webhook ID movie_night_on. Actions for Movie Night:
A
Turn on TV and switch to HDMI 2
media_player.turn_on → TV
media_player.select_source → TV → HDMI 2
B
Turn on the Yamaha receiver
media_player.turn_on → Yamaha
C
Dim the lights
light.turn_on → your light entity → Brightness: 10%
Optional Power Off webhook: Create a third automation with Webhook ID all_off and add turn_off actions for the TV, Yamaha, and Sonos. Tap Power Off in the app to trigger it.
What is an entity ID?
Every device in Home Assistant has a unique name called an entity ID. It always looks like type.device_name — for example media_player.sonos_soundbar or light.living_room. This app needs the exact ID so it can control the right device.
How to find your entity IDs
1
Open Developer Tools
In HA, click Developer Tools in the left sidebar (the icon looks like </>). If you don't see it, go to Settings → System → enable Developer mode first.
2
Click the States tab
You'll see a searchable list of every device. In the search box type the name of your device, like sonos or yamaha.
3
Copy the entity ID from the left column
The first column shows the entity ID. It will look like media_player.sonos_beam or media_player.yamaha_rx_v685. Copy it exactly, including the prefix (media_player., light. etc).
4
Paste it into this app
Replace the placeholder in Settings or the wizard field with the exact ID you copied.
Common entity types you need:
• TV → media_player.your_tv_name
• Sonos → media_player.sonos_name
• Yamaha → media_player.yamaha_name
• Fire TV → media_player.fire_tv_name
• Lights → light.room_name
Can't find a device? Make sure it's powered on when you search — some devices only appear in HA when they're active. Also check that the integration (Sonos, Fire TV etc) is installed in HA Settings → Integrations.
Enable Kodi's remote control
1
Open Kodi on your PC
Launch Kodi. Go to Settings (the gear icon on the home screen).
2
Navigate to Services
Settings→ Services→ Control
3
Enable HTTP control
Turn on Allow remote control via HTTP. Set the port to 8080. Username and password are optional — leave blank for easier setup on a home network.
4
Find your PC's local IP
On the PC: press Win + R, type cmd, press Enter. In the black window type ipconfig and press Enter. Look for IPv4 Address — it'll be something like 192.168.1.45.
5
Enter the URL in this app
In Settings → Kodi & PC Agent, set Kodi URL to: http://192.168.1.45:8080/jsonrpc (replace with your actual IP). Tap Ping Kodi to test.
Library scan: After setting up Kodi, point it at your movie folder: Kodi → Settings → Media → Library → Add Videos → browse to your external drive. Once scanned, movie posters and info appear automatically in this app's Now Playing bar.
Download the PC Agent files
Save both files to the same folder on your PC, e.g. C:\MovieRoom\. Then follow the steps below.
What is the PC Agent?
The PC Agent is a tiny program that runs on your gaming PC in the background. It lets this app launch programs (like Kodi or Chrome) and send keystrokes to the PC — like a remote control for Windows. It's a single Python file that starts automatically when you log in.
Setup (one time, 5 minutes)
1
Install Python on your PC
Go to python.org/downloads and download Python 3. During install, check the box that says Add Python to PATH — important!
2
Copy the agent files to your PC
Download pc-agent.py and install-agent.bat from this project (they're included in the download). Put them in a folder like C:\MovieRoom\.
3
Run the installer
Right-click install-agent.bat → Run as administrator. It installs a dependency and registers the agent to start automatically when you log in.
4
Find your PC's IP and enter it here
Press Win + R → type cmd → type ipconfig. Copy the IPv4 address (e.g. 192.168.1.45). In Settings → Kodi & PC Agent, set PC Agent URL to http://192.168.1.45:9876. Tap Ping Agent to confirm.
⚠️ Not connecting? Windows Firewall is the most common cause. Run install-agent.bat as Administrator — it adds the firewall rule automatically. Or manually: Windows Defender Firewall → Advanced Settings → Inbound Rules → New Rule → Port → TCP 9876 → Allow.
Customize app shortcuts: Open pc-agent.py in Notepad and edit the APP_MAP section near the top to point to wherever your apps are installed.
Security note: The PC Agent only accepts connections from your local network. Don't forward port 9876 through your router to the internet.
